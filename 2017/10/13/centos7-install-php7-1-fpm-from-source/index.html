FAQ

如果不需要 ldap 模块可以取消，因为在编译过程中可能会出现两个错误

1) 提示找不到 ldap 库文件，解决办法就是将 /usr/lib64/ 目录下的 libldap 库文件拷贝到 /usr/lib/ 目录下

2) 进行编译时系统抛出错误提示

解决办法编辑 Makefile (vim Makefile) 文件，找到 EXTRA_LIBS = 字段，在尾部追加 -llber 保存退出，继续执行 make && meke install
