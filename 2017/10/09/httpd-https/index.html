什么时候使用 301,302 呢？

如果你有一个老地址已经确定不实用了话建议使用 301，像 http 转 https 这种，因为 http 协议的 URL 地址还会使用到，所以一般使用 302 重定向，因为这会关系到搜索引擎优化问题
